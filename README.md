# [Web Animeh](https://github.com/NathanaelGT/web-animeh)

# Requirement

- [Git](https://git-scm.com/) (dev)
- [Bun](https://bun.sh)
- [FFmpeg](https://www.ffmpeg.org)

# Installation

```shell
$ git clone https://github.com/NathanaelGT/web-animeh
$ cd web-animeh
$ bun install
```

# Run Dev Server

```shell
$ bun dev
```

# Build

```shell
$ bun run build
```

# Run Production Server

```shell
$ bun ./dist
```
